# Expense-Tracker
A simple Python Expense Tracker that allows users to enter multiple expenses, calculates the running total, and displays the total amount spent.

This project is a basic Expense Tracker built with Python as part of the @DecodeLabs Internship Program. The application allows users to continuously enter expense amounts, automatically adds them to a running total, and displays the final amount spent when the user exits the program. The project demonstrates the use of loops, user input, conditional statements, and accumulator logic.
